# 重要元件的功能架構

### App.vue
- 作為網站根元件
- 使用 <Toast> 去顯示操作狀態 (主要為 api call 的 resp)
- 適合開發偵錯用的 <JSONView> 元件
- 透過 vueuse 斷點調整網頁左右 padding
- 樣式 CSS 的全域變數

### Header.vue
- 下滑線展開動畫

## inMenubarEnd.vue
**前端部分**:
- 包含三個重要按鈕區塊: 用戶管理、購物車、like商品列表
- 用戶管理: OAuthLogin.vue
- 購物車:
  - 點擊跳轉到購物車列表
  - 購物車圖標，應顯示購物車數量狀態
- like商品列表: 借商品列表的顯示方式，數據取得則批次依據商品id來取得 🌙🌙🌙

**supabase部分**:
- 配置購物車的 spec 🌙🌙🌙
  - 應採取「事件溯源系統」的權限配置
- like商品儲存商品id陣列即可 🌙🌙🌙

### OAuthLogin.vue
- 用戶登入/登出操作
- supabase REST API google 第三方登入驗證流程
  - token 儲存在 localStorage 作持久化管理，登出時主動刪除
- 用戶管理頁面跳轉

### ProductsDisplayBody.vue
重要元件: 左側欄導航、搜尋欄、右側商品列表頁面

### ProductList.vue
- 輸入: 路由、商品數據
- 路由觸發 get 商品類別列表結果
- 路由觸發 get 商品搜尋結果
- 路由觸發 get like列表，在批次取得 like 數據

### SideNavbar.vue
- 類別導航部分: 桌面版與非桌面版使用不同元件顯示
- 搜尋欄: 可以紀錄用戶輸入的歷史紀錄
